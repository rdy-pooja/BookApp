# BookApp

ASP.NET WEB API BookApp for Performing Below Functions:
  1. Post Entity (Book)
  2. Get Book from ID
     #Note - ID is generated as part of Post Operation.
  3. Get Books
  4. Modify Book Details

Visual Studio 2019
Database Used - MSSQL

Refer SQLHealthCheck.cs file for the SQL Connectivity Health.
